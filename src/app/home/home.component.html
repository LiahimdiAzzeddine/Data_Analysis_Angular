Bienvenue dans la partie admin

Avec cette application vous pouvez consulter, créer, modifier et supprimer des datasets

Cette application a été programmée avec Spring boot, Angular et MongoDB

 Information

  Contexe , Dataset

 Cette application et la partie adminstrateur d’une application web distribuée basée sur la Technologie spring architecture
 micro-services, et flutter pour l’analyse, la visualisation et le Clustering des données.

 les données Utilisees dans cette application sont collectees depuis plusieurs sites web de vendre
 des Appartement dans la ville de Tanger.

 mais aussi il ya des données collectees a partir des instances de l'application client flutter concernant les appareils utiliser et aussi
 les information personelles fournir par les utilisateurs.

 l'utilisateur de cette application peut créer des modèles d'apprentissage automatique K-means,SOM et EN.
